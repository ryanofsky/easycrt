Want to learn EasyCRT?

  • First, check out the Command Reference. Although a lot of it won't make much sense at first, you can get a better idea of what the features are and what EasyCRT can do. You can also look through the rest of the web site (which was originally made to present EasyCRT at a science fair).

  • Then look at the Demo Programs. They should be in C:\TPW\EasyCRT\samples. There are only a few right now but there will be more soon. They can show you how to use EasyCRT commands. Here is the list so far -

emouse.pas - This program shows how to get mouse input.

getpos.pas - This program shows how to use the getpos command to return information about the window

howlong.pas - Tells you how long windows has been running.

inkey.pas - Retrieves virtual key codes, demonstrates inkey usage.

inkeyasc.pas - Lets you type on screen, shows how inkeyasc can be used to input text.

title.pas - Advanced use of WinCRT graphics. See for yourself.

stick.pas - Computer game made last year with EasyGDI. Currently uses EasyGDI 1.0, I will port it over to 2.0 soon.
