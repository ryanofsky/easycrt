  • First, check out the Command Reference. Even if it doesn't make sense at first, you can get an idea of what EasyCRT can do.

Understanding the Command Reference

The command reference is a listing of the declarations used to create EasyCRT's constants, variables, types, and functions mixed in with explanations of what all of these things do.

The declarations tell you what syntax to use with the commands. For example,

procedure settitle(lbl:string);

tells you that settitle is a procedure that has a single argument that is a string. You would use it in your program like:

settitle('My Title');
