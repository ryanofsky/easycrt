Date Program Version Description
October '97 Area! 1.0 A computer game teaching how to find areas of circles, triangles and rectangles. This was where I was first figuring out the graphics commands.
October '97 EasyCRT 1.0 After my computer teacher saw the some of the graphics in my program he asked if I could make add some graphics routines to WinCRT. These were very simple and it was still very buggy and clumsy. There were five new commands: Line, Circle, Drawpicture, Setfont and Pprint. These commands created and deleted pens and the device contexts with their procedures. So the command lines were really long and they were slower. They were also unpredicatable because they completely ignored the brush.
December '97 EasyCRT 2.0 This added most of the new drawing commands in it today and was completely backwards compatible with 1.0, but the new inkey and inkeyasc functions were still a little buggy, and there was no mouse or window functions.
December '97 EasyCRT 2.01 Fixed the inkey and inkeyasc functions. (the computer projects for my class were made mostly in this version.
January '98 EasyGDI 1.0 I wanted to give my game, Stickfighter, some features which would not be possible using EasyCRT. These included having more control over the window (always on top, no border, centered, etc), getting mouse input, and drawing in memory then copying to the screen rather than just drawing to screen. I also wanted to be able to use and learn ObjectWindows. But since I didn't want to have to use the GDI drawing commands I made EasyGDI. This allowed me to have the best of both worlds.
January '98 EasyCRT 3.0 This added mouse input, window control, and File Dialog Box, features to EasyCRT. However, by this time we had finished our projects it was not really used by anybody.
March '98 EasyCRT 3.2 This was made specifically for this Science Fair.It eliminated some of the old commands and made certain parts more efficient.
November '98 EasyCRT 4.0 This version eliminated all of the known bugs in the previous versions and added the ability to use colored text in the CRT window. Many of the commands have been renamed and redesigned so it is more efficient but no longer compabitble with older versions of EasyCRT.
November '98 EasyGDI 2.0 This version features a complete redesign of the EasyGDI interface which is faster, easier to use, and filled with more features.
