Command Demos - these were made to demonstrate how to use some of EasyCRT's commands in a program.

  • Mouse Demo - uses MOUSEX, MOUSEY, LDOWN, RDOWN. Hit any key to exit.
  • Inkey - shows how to use inkey, Click the screen to exit.
  • Inkeyasc - shows how to use inkeyasc. Click the screen to exit.
  • Plusmove - originally made as a simple demonstration of object oriented programming, but also makes good use of the inkey command. Move the plus sign around the screen with the arrows and click the screen to exit.
